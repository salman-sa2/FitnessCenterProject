# Fitness Center Randevu ve Yönetim Sistemi

Bu proje, bir veya birden fazla spor salonu için randevu ve yönetim sisteminin ASP.NET Core MVC kullanılarak geliştirilmesini amaçlamaktadır. Sistem üzerinden üyeler kayıt olabilir, hizmet (fitness, yoga, PT vb.) seçerek antrenörlerden uygun saatlerde randevu alabilir ve randevularını görüntüleyebilir.

Yönetici (Admin) paneli ile:
- Salonlar, hizmetler ve antrenörler yönetilebilir,
- Antrenörlerin uygunluk saatleri tanımlanabilir,
- Üye randevuları görüntülenip onaylanabilir veya iptal edilebilir.

Projede ASP.NET Core MVC, Entity Framework Core, LINQ, SQL Server ve Bootstrap 5 teknolojileri kullanılmaktadır. Ek olarak, kullanıcıların boy, kilo ve hedef bilgilerine göre temel egzersiz/öneri sunan basit bir yapay zeka entegrasyonu bulunmaktadır.
